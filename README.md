# Flujo en Redes — Notas de Aula

Notas de aula del curso de **Optimización en Flujo en Redes**, Departamento de Ingeniería Industrial, Universidad de los Andes.

**Profesores:** David Álvarez-Martínez y Alejandra Tabares

---

## Contenido

| Semana | Tema |
|--------|------|
| 1 | Introducción, motivación y conceptos de grafos |
| 2 | Herramientas computacionales — Gurobi |
| 3 | Redes como Programas Lineales |
| 4 | Método Símplex en Flujo en Redes |
| 5 | Problema de Transporte |
| 6 | Problema de Asignación |

## Publicación

Las notas están publicadas como un Jupyter Book en GitHub Pages. Cada vez que se hace `push` a `main`, el sitio se reconstruye automáticamente.

## Uso local

```bash
pip install jupyter-book
jupyter-book build .
# Abrir _build/html/index.html
```
